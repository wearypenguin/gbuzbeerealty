# Image Assets for G. Buzbee Realty

This directory contains all images for the website.

## Directory Structure

```
images/
├── properties/          # Property photos
├── blueprints/          # Blueprint and plan images
├── logo/               # Company logo files
└── placeholders/       # Temporary placeholder images
```

## Image Guidelines

### Property Photos
- **Format:** JPG or WebP
- **Size:** Max 1920px wide
- **Optimization:** Compress before uploading
- **Naming:** Use descriptive names (e.g., `123-main-st-exterior.jpg`)

### Blueprints
- **Format:** JPG, PNG, or PDF
- **Size:** High resolution for detail visibility
- **Naming:** Include property address or plan name

### Logo
- **Formats needed:**
  - SVG (for scalability)
  - PNG (with transparent background)
  - Sizes: 64x64, 128x128, 256x256

## Accessibility Notes

All images should have descriptive alt text added in the HTML. Photos of properties should describe key features visible in the image.
